# Поддерживаемые платформы

Media Downloader поддерживает 20+ платформ для скачивания аудио и видео контента.

## Видео платформы

### YouTube
- **Форматы URL**: `youtube.com/watch?v=*`, `youtu.be/*`
- **Возможности**: Все качества (144p-4K), извлечение аудио
- **Ограничения**: Контент с возрастными ограничениями может требовать аутентификацию

### Vimeo
- **Форматы URL**: `vimeo.com/*`
- **Возможности**: Поддержка HD и 4K
- **Ограничения**: Приватные видео не поддерживаются

### Dailymotion
- **Форматы URL**: `dailymotion.com/video/*`, `dai.ly/*`
- **Возможности**: Множество вариантов качества
- **Ограничения**: Нет

### Rutube
- **Форматы URL**: `rutube.ru/video/*`
- **Возможности**: Поддержка российской видеоплатформы
- **Ограничения**: Нет

### Newgrounds
- **Форматы URL**: `newgrounds.com/*`
- **Возможности**: Анимация и видео контент
- **Ограничения**: Нет

## Социальные сети

### TikTok
- **Форматы URL**: `tiktok.com/@*/video/*`, `vm.tiktok.com/*`
- **Возможности**: Видео с водяным знаком или без
- **Ограничения**: Могут применяться региональные ограничения

### Instagram
- **Форматы URL**: `instagram.com/p/*`, `instagram.com/reel/*`
- **Возможности**: Посты, Reels, Stories
- **Ограничения**: Только публичный контент

### Facebook
- **Форматы URL**: `facebook.com/*/videos/*`, `fb.watch/*`
- **Возможности**: Публичные видео
- **Ограничения**: Приватные видео не поддерживаются

### Twitter / X
- **Форматы URL**: `twitter.com/*/status/*`, `x.com/*/status/*`
- **Возможности**: Видео твиты
- **Ограничения**: GIF не поддерживаются

### VK (ВКонтакте)
- **Форматы URL**: `vk.com/video*`, `vk.ru/video*`
- **Возможности**: Множество вариантов качества
- **Ограничения**: Нет

### Одноклассники
- **Форматы URL**: `ok.ru/video/*`
- **Возможности**: Поддержка российской социальной сети
- **Ограничения**: Нет

### Reddit
- **Форматы URL**: `reddit.com/r/*/comments/*`, `redd.it/*`
- **Возможности**: Видео посты
- **Ограничения**: Нет

### Tumblr
- **Форматы URL**: `tumblr.com/*`
- **Возможности**: Видео посты
- **Ограничения**: Нет

### Snapchat
- **Форматы URL**: `snapchat.com/*`
- **Возможности**: Публичные истории
- **Ограничения**: Приватный контент не поддерживается

## Особенности платформ

### YouTube
- Поддерживает контент с возрастными ограничениями с аутентификацией
- Поддержка плейлистов скоро появится
- Прямые трансляции не поддерживаются

### TikTok
- Удаление водяного знака зависит от доступности
- В некоторых регионах могут быть ограничения

### Instagram
- Stories истекают через 24 часа
- Приватные аккаунты не поддерживаются

### Facebook
- Только публичные видео
- Прямые трансляции не поддерживаются

### Twitch
- Только клипы, не VOD или прямые трансляции
- Поддержка VOD планируется

### SoundCloud
- Только бесплатные треки
- Premium контент не поддерживается

## Техническая реализация

Все платформы используют один из этих методов:

1. **Официальное API**: Прямая интеграция с API (YouTube, SoundCloud)
2. **Web Scraping**: Парсинг HTML/JavaScript
3. **Reverse Engineering**: Анализ мобильного API
4. **yt-dlp Fallback**: Универсальный экстрактор как резервный вариант
