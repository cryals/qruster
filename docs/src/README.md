# Media Downloader Documentation

Welcome to the Media Downloader documentation. This is a universal selfhosted service for downloading audio and video content from 20+ popular platforms.

## Quick Links

- **English Documentation**: Start with [Concept & Ideas](en/concept.md)
- **Русская документация**: Начните с [Концепция и идеи](ru/concept.md)

## Features

- Download audio in multiple formats (MP3, OGG, WAV)
- Download video in various qualities (144p to 4K)
- Support for 20+ platforms including YouTube, TikTok, Instagram, VK, Bilibili
- Material Design 3 interface
- Selfhosted and privacy-focused
- Docker-ready for easy deployment

## Technology Stack

- **Frontend**: React + Material-UI (MUI) v6
- **Backend**: Rust (Axum framework)
- **Containerization**: Docker + Docker Compose
- **Reverse Proxy**: Caddy with automatic SSL
- **Documentation**: mdBook
