# Руководство пользователя

## Начало работы

После установки вы можете начать использовать Media Downloader для скачивания аудио и видео с поддерживаемых платформ.

## Базовое использование

### 1. Запуск приложения

**Режим разработки:**
```bash
./scripts/run.sh
# Выберите опцию 1
```

**Режим продакшена:**
```bash
./scripts/run.sh
# Выберите опцию 2
```

### 2. Доступ к веб-интерфейсу

Откройте браузер и перейдите по адресу:
- Разработка: http://localhost:3000
- Продакшен: https://ваш-домен.com

### 3. Скачивание медиа

1. **Вставьте URL**: Скопируйте URL видео или аудио с любой поддерживаемой платформы
2. **Дождитесь анализа**: Система автоматически определит платформу и извлечет доступные форматы
3. **Выберите формат**: Выберите предпочитаемое качество и формат
4. **Переключите "Только аудио"**: Включите, если нужно только аудио
5. **Нажмите "Скачать"**: Ваш файл будет подготовлен и скачан

## Опции скачивания

### Качество видео

Доступные качества зависят от источника:
- **4K** (2160p) - Ультра HD
- **1080p** - Full HD
- **720p** - HD
- **480p** - SD
- **360p** - Низкое качество
- **144p** - Мобильное качество

### Аудио форматы

- **MP3** - Универсальная совместимость (128-320 kbps)
- **OGG** - Открытый формат, хорошее качество
- **WAV** - Без потерь, большой размер файла

### Видео форматы

- **MP4** - Наиболее совместимый
- **MKV** - Высокое качество контейнер
- **WEBM** - Оптимизирован для веба

### Специальные опции

- **Только аудио**: Извлечь только аудио дорожку
- **Без звука**: Скачать видео без звука
- **Лучшее качество**: Автоматически выбрать наивысшее доступное качество

## Продвинутые функции

```
https://www.instagram.com/reel/REEL_ID/
```

## Советы и лучшие практики

### Для лучших результатов

1. **Используйте прямые ссылки**: Избегайте сокращенных URL когда возможно
2. **Проверяйте качество**: Просмотрите доступные форматы перед скачиванием
3. **Извлечение аудио**: Используйте режим только аудио для музыки чтобы сэкономить трафик
4. **Стабильное соединение**: Обеспечьте стабильный интернет для больших файлов

### Производительность

- **Большие файлы**: Скачивание файлов более 500MB может занять несколько минут
- **Часы пик**: Некоторые платформы могут быть медленнее в часы пик
- **Повтор**: Если скачивание не удалось, подождите момент и попробуйте снова

### Приватность

- **Без отслеживания**: Мы не отслеживаем ваши скачивания
- **Без хранения**: Файлы временно хранятся и удаляются через 1 час
- **Локальная обработка**: Вся обработка происходит на вашем сервере

## Решение проблем

### "Не удалось извлечь информацию о медиа"

**Возможные причины:**
- Неверный URL
- Приватный или ограниченный контент
- Платформа временно недоступна
- Проблемы с сетью

**Решения:**
1. Проверьте правильность и доступность URL
2. Проверьте, является ли контент публичным
3. Попробуйте снова через несколько минут
4. Проверьте интернет соединение

### "Скачивание не удалось"

**Возможные причины:**
- Файл слишком большой (лимит >2GB)
- Недостаточно места на диске
- Таймаут сети
- Блокировка платформы

**Решения:**
1. Попробуйте более низкое качество
2. Проверьте доступное место на диске
3. Повторите скачивание
4. Свяжитесь с администратором если проблема сохраняется

## Ограничения

### Текущие ограничения

- Максимальный размер файла: 2GB
- Максимальная длительность видео: 3 часа
- Нет поддержки плейлистов (скоро)
- Нет скачивания прямых трансляций
- Ограничение скорости: 5 скачиваний в минуту на IP
